
## nigo-vue-tab v1.1.1
`    npm install nigo-vue-tab
`
## Example Usage
`    import {Tab,TabItem} from 'nigo-vue-tab'
`


```
 <Tab :content="[1,2,3,4,5]" 
       tab-width="800"
       tab-height="500"
       theme-color="#333"
<TabItem>1</TabItem>
<TabItem>2</TabItem>
<TabItem>3</TabItem>
<TabItem>4</TabItem>
       ></Tab>
```
